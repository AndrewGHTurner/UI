# Ways to use this ui library

## Ways to integrate this library into a project

### Static inclusion

### Dynamic linking

#### Procedure:

1. go to project properties -> linker -> input -> additional dependencies and add the UI.lib file (UI-d.lib for debug mode)
2. go to project properties -> c/c++ -> general -> additional include directories and add the path to the UI header files


## Types of produced artefact

### Executible (.exe)

### Dynamic link library (.dll)

### Interpreted UI definition files (similar to HTML/CSS)
